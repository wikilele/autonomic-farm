# Autonomic Farm Framework

Parallel and Distributed Systems final project

## Usage

```
    git clone https://github.com/wikilele/autonomic-farm.git
    cd autonomic-farm
    chmod +x build.sh
    chmod +x run.sh
    chmod +x plot.sh

    ./build.sh
    ./run.sh

    # to plot graph-cli is needed
    head plot.sh
    ./plot.sh
```
